format/2

format(Format::string(), Args::list()) -> string()

Format: format string
Args: format argument

returns: string

Format string and data to a string. Approximates features of OTP io_lib:format/2, but only supports ~p and ~n format specifiers. Raises bad_format error if the number of format specifiers does not match the length of the Args.
